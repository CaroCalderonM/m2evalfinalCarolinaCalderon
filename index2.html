    Git is a distributed version control system that enables developers to track and manage changes in a project’s source code. It is essential for collaboration on software development projects, where multiple programmers can work simultaneously on different parts of a project. Git is widely used in software engineering, web development, and document version control.

    GitHub is a web-based platform that uses Git to provide a collaborative environment where developers can host, review, and manage software projects. It offers features such as version control, issue tracking, and code review. GitHub is fundamental in software development, especially in open-source projects and collaborations within distributed teams.

    HTML (HyperText Markup Language) is the standard markup language used to create the structure and content of web pages. It defines the layout of text, images, links, and other multimedia elements on a page. HTML is essential in web development and forms the foundation upon which all web applications and sites are built.

    CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación visual de un documento HTML. CSS permite a los desarrolladores controlar el diseño, colores, tipografía y disposición de los elementos en una página web. Se utiliza ampliamente en el diseño web para mejorar la apariencia y usabilidad de sitios y aplicaciones web.

    JavaScript is an interpreted programming language that enables developers to create interactivity and dynamism on web pages. It is crucial in frontend development, allowing manipulation of the DOM, form validation, and the creation of interactive web applications. JavaScript is also the foundation of many modern frameworks and libraries such as React, Angular, and Vue.js.

    PostgreSQL is an open-source relational database management system known for its robustness and extensibility. It offers advanced features such as support for transactions, referential integrity, and stored procedures. PostgreSQL is widely used in enterprise applications, data analysis, and environments where efficient and secure handling of large volumes of data is required.

    R is a programming language and software environment primarily used for statistical analysis and data visualization. It offers a wide variety of tools and libraries for performing quantitative analysis, statistical modeling, and data mining. R is widely used in fields such as biostatistics, scientific research, and financial data analysis.

    UX/UI Design refers to the discipline focused on creating digital products that provide optimal user experiences (UX) and attractive, functional user interfaces (UI). UX/UI design is crucial in the development of applications and websites, ensuring that they are intuitive, accessible, and aesthetically pleasing for end users.

    Figma is a cloud-based design tool that allows teams to create, prototype, and collaborate on user interface designs. It offers vector design features, interactive prototyping, and real-time collaboration. Figma is widely used in UX/UI design, especially by distributed teams that require a collaborative platform.

    Photoshop is an image editing software developed by Adobe, known for its wide range of tools for manipulating and creating raster graphics. It is used in graphic design, photography, and multimedia production, allowing users to retouch photos, create digital illustrations, and design complex visual interfaces.

    Photopea is a web-based image editing tool that offers functionalities similar to Adobe Photoshop. It supports a wide range of file formats and allows advanced image manipulation and graphic design. Photopea is a popular choice among designers seeking a free and accessible alternative for image editing without the need to install software.

    Freehand is a vector design application that allows users to create digital illustrations, graphics, and layouts with scalable precision. It was widely used for graphic design and layout before being discontinued. It remains relevant in the history of graphic design and in studies of classic digital illustration tools.

    CANVA is an online graphic design platform that allows users to create graphics, presentations, posters, and other visual content easily. It offers customizable templates and an intuitive interface, making it ideal for users with no prior graphic design experience. CANVA is used in digital marketing, social media content creation, and the design of printed materials.

    Kittl is an online graphic design tool specializing in the creation of vector graphics and typography. It offers an intuitive set of tools for designing logos, posters, and other graphic elements with precision and ease. Kittl is popular among designers and creatives looking for an accessible platform for high-quality design without the need for complex software.

    ChatGPT is a language model based on artificial intelligence developed by OpenAI, designed to generate coherent and relevant text in response to user input. It is used in a variety of fields, including customer service, content generation, online tutoring, and automation of natural language processing tasks.

    Gemini AI is an advanced artificial intelligence developed by Google, designed to offer enhanced capabilities in natural language processing, contextual understanding, and content generation across multiple domains. It is applicable in areas such as virtual assistance, data analysis, and process automation in various industries.

    Copilot AI is an artificial intelligence tool integrated into software development environments, such as GitHub Copilot, that assists programmers by suggesting lines of code and completing repetitive tasks. It is used in software development to improve productivity, reduce errors, and speed up the coding process.

    Copilot Image AI is an AI-powered tool designed to assist designers and content creators in generating images and graphics. It facilitates visual content creation by automating tasks and suggesting design elements, making it useful in digital marketing, graphic design, and multimedia production.

    DALL-E Image AI is an artificial intelligence model developed by OpenAI that generates images from textual descriptions. It is capable of creating illustrations, artistic concepts, and complex graphics, and is used in areas such as advertising, digital content creation, and creative design.

    Leonardo Image AI is an artificial intelligence platform specialized in generating high-quality images from detailed descriptions. It is used in graphic design, digital art, and visual content creation for applications such as video games, advertising, and media.

    Ideogram Image AI is an artificial intelligence tool that generates images based on complex patterns and structures. It is useful in graphic design, abstract art creation, and the development of unique visual elements for branding and marketing. As of July 2024, it is considered the best artificial intelligence for creating images that include text, in addition to generating both image files and vectors.

    Excel is a spreadsheet developed by Microsoft that allows data manipulation, statistical analysis, and information visualization through charts. It is widely used in business, finance, research, and project management for organizing and analyzing large volumes of data.

    Word is a word processor by Microsoft used for creating, editing, and formatting documents. It is essential in drafting reports, proposals, and any type of textual document in professional and academic settings.

    PowerPoint is a presentation application developed by Microsoft, used to create visual presentations with slides that can include text, images, charts, and multimedia. It is widely used in conferences, business meetings, and educational settings to effectively communicate ideas and data.

    OneDrive is a cloud storage service developed by Microsoft that allows users to securely store, sync, and share files. It is used in both personal and business environments to access documents from any internet-connected device and collaborate in real-time. OneDrive is essential for file management and remote collaboration, enabling teamwork and project continuity without the need for physical infrastructure.

    SharePoint is a collaboration and content management platform developed by Microsoft, which facilitates the creation of internal websites and the organization of information, documents, and business processes. It is widely used in corporate environments for project management, document storage, and workflow automation, improving efficiency and communication within organizations.

    Acrobat Pro Editing Mode is an advanced version of Adobe Acrobat software that allows editing, converting, and manipulating PDF documents. It includes tools for adding and modifying text, images, and links, making it essential for editing official documents and preparing professional reports.

    Advanced Electronic Signature is a digital authentication method that ensures the identity of a signer through cryptography. It is used in the legal validation of electronic documents, ensuring the integrity and authenticity of contracts, agreements, and other important documents in sectors such as banking, government, and business.

    SAP HANA and SAP are enterprise software platforms developed by SAP, where SAP HANA is an in-memory database that provides real-time data processing capabilities, and SAP is a suite of enterprise resource planning (ERP) solutions. These tools are fundamental in managing business operations, from finance and logistics to human resources and sales, facilitating data-driven decision-making.
